# CV Digital - Fiamma Loreto Tamayo Villarroel

Este proyecto corresponde a un CV digital responsivo desarrollado con HTML, CSS, Bootstrap y JavaScript.  
El objetivo es presentar información profesional de manera clara, moderna y accesible desde cualquier dispositivo.

## Características
- Diseño responsivo con Bootstrap
- Navbar con navegación por secciones
- Modo oscuro
- Cambio de colores
- Animaciones al hacer scroll
- Formulario con validación en tiempo real
- Uso de HTML semántico
- Código organizado en archivos separados

## Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- jQuery
